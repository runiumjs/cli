# @runium/cli

[![npm](https://img.shields.io/npm/v/@runium/cli)](https://www.npmjs.com/package/@runium/cli)
[![license](https://img.shields.io/npm/l/@runium/cli)](LICENSE)

Command-line interface for Runium.

## Installation

```bash
npm install -g @runium/cli
```

or

```bash
yarn global add @runium/cli
```

## Usage

```bash
runium [command] [options]
```

## Requirements

- Node.js >= 20.12.0
